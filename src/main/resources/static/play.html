Tetris-Combat is a real-time multiplayer game based on the original game designed in 1984 by Alekséi Pázhitnov. The game manages a user base that registers to play in rooms created by themselves where they can meet and play with friends. Additionally, each user can consult the statistics of their profile, games played and account. General Idea
Played with 7 randomly generated figures, these begin to fall one by one while the player must decide the rotation of the piece while it falls, as well as where it should fall. When a horizontal line is completed, that line disappears (generating an score to the player) and all the pieces above descend one position. The fall of the pieces accelerates progressively, the game ends when the pieces are piled up to the top.

Real-Time Orientation
Developed mainly in the multiplayer mode that offers where we find an architecture based on sockets allowing real-time communication between two instances of our application.
